plivo

docs

Plivo provides you with an array of tools to enable you to create powerful telephony apps without having to worry about the nuts and bolts. Make and receive calls, send texts, create conferences and more.

Our APIs are used in conjunction with XML responses to control the flow of a call or a message. Plivo also provides helper libraries in various languages such as PHP, Python, Ruby, and Java. To learn more, click on one of the links below.

API

Plivo's APIs provide functions that enable your web application to perform telephony operations.

XML

A pre-defined set of XML elements that the application must use to control the progress of the operation.

Helpers

Helper libraries simplify the process of interacting with Plivo. All libraries are open sourced and hosted on Github.

SDKs

Plivo SDKs let you integrate voice natively in the browser and mobile devices.
